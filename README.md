# project-portfolio-demo

"空杯"独立开发项目作品集，涵盖网页应用、互动游戏、3D 可视化、独立站主题与 Godot 原型，用于展示完整项目策划、前端实现、玩法系统、视觉资产与工程组织能力。

## 项目内容

- `个人成长工作台`：React + Vite 构建的个人成长与目标管理工具。
- `宇宙模拟探索网页`：基于 Three.js 的银河系 3D 探索与科普交互页面。
- `回声深渊`：原生 Web 技术实现的横版动作探索游戏原型。
- `宠物用品-电商独立站`：面向宠物用品品牌的 Shopify 主题与静态预览。
- `打砖块`：单文件 HTML 打砖块小游戏。
- `绯红回响（Crimson）`：Godot 独立游戏原型，包含策划文档、场景、脚本与美术资源。
