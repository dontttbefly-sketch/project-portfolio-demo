# 回声深渊 / Echo Abyss

一款 100% 由代码合成像素美术的 2D Roguelite + Metroidvania 探索平台游戏。  
零依赖、零素材、零安装。

## 9 个主题房间 + 三层难度

地图由 **9 个手工模板**驱动（每个 zone 3 个），每间房一个标志性 gimmick：

| Zone | 模板 | gimmick |
|---|---|---|
| 1 起源洞窟 | 入门走廊 / 碎墙之路 / 第一篝火 | 跳跃节奏 / 蓄力破墙 / 二段跳试用 |
| 2 灼热深渊 | **回声桥** ⭐ / 熔岩冲刺 / 炼铸竞技场 | **强制留回声当桥** / 冲刺追逐 / 三路线综合 |
| 3 湮灭神殿 | 镜厅 / **暗梯井** ⭐ / **倒置之塔** ⭐ | 镜像幽灵 / **回声路标 + 黑暗** / **重力翻转 + 回声链** |

⭐ = 必须使用回声机制才能通过的房间。

**能力解锁顺序**（不再默认全开）：
- Zone 1：仅单跳 + 下劈
- Zone 1→2 过渡圣坛：解锁二段跳
- Zone 2 中段圣坛：解锁冲刺
- Zone 3 圣坛：解锁抓墙

**可达性保证**：每个生成的房间都过 BFS 验证 — spawn 到所有出口/圣坛在当前 zone 能力下必可达，否则触发 fallback 修复。

## 核心创意：回声系统 ⭐

每次死亡会在死亡地点留下一个**幽灵回声**，跨周目持久存在（localStorage）。  
靠近回声按 **E** 可以选择四种互动：

| 互动 | 效果 | 代价 |
|---|---|---|
| **倾听** | 听它讲述死因，高亮房内陷阱 | 无 |
| **吸收** | 永久消耗，获得对应抗性 buff | 回声消失 |
| **召唤** | 复活成幽灵盟友，战斗 15 秒 | 若幽灵阵亡，回声永久消失 |
| **渡桥** | 与之对话 3 秒（按住 E） → 化为实体平台 | 渡桥时无法移动 |

**进阶**：长时间未访问的回声会被深渊腐蚀，化为「堕落回声」敌人。击败后取回死亡时丢落的全部碎片。

## 运行方式

由于使用 ES6 modules，需要通过 HTTP 服务（不能直接双击 `file://` 打开）。

```bash
# 进入项目目录
cd 2D

# 任选一种启动本地服务器
python3 serve.py 8088              # 推荐 — 自带 no-cache 头，开发期改代码立即生效
# 或
python3 -m http.server 8088
# 或
npx serve -p 8088
```

然后浏览器打开 `http://localhost:8088`

## 操作

| 键位 | 动作 |
|---|---|
| **A / D** | 左右移动 |
| **K / 空格** | 跳跃（空中再按 = 二段跳 *需解锁*） |
| **S + K** | 下穿单向平台 |
| **J** | 攻击（连续按 → 三连击）|
| **长按 J** | 蓄力重斩（可破脆弱墙）|
| **U** | 远程魔法弹（耗法力）|
| **L** | 冲刺（前 0.16s 无敌帧 *需解锁*）|
| **I** | 格挡（0.16s 内挡下 = 完美格挡反震）|
| **E** | 互动（回声 / 篝火 / 升级祭坛）|
| **F** | 召唤 — 直接复活最近回声为幽灵 |
| **H** | 喝治疗瓶（地面上）|
| **TAB** | 切换全屏地图（显示所有房间与连接）|
| **ESC** | 暂停 |

## 玩法亮点

- **硬核战斗**：体力条 / 攻击 telegraph / 完美格挡 / 冲刺无敌帧 / 击退反馈
- **程序化关卡**：每次新游戏 11+ 个房间随机生成，3 个生物群落（森林 / 熔岩 / 神殿）
- **Metroidvania 解锁**：二段跳、抓墙跳、下劈、冲刺无敌帧
- **隐藏房间**：脆弱墙后藏着升级祭坛
- **永久升级**：祭坛能永久增加生命上限 / 治疗瓶上限
- **多阶段 Boss**：HP 50%、25% 触发新阶段（弹幕 / 召唤）
- **检查点篝火**：休息 = 满血 + 补瓶 + 敌人重生 + 保存进度

## 美术

**所有像素 sprite、地形、粒子、UI 都在代码里实时合成**。  
零图片资源，全部用 ASCII 模式 + 16 色调色板渲染。  
背景视差山脉用纯数学填色 + 噪声生成。  
音效用 Web Audio API 合成方波 + 噪声实现 8-bit 风格。
