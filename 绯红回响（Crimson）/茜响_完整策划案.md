## 项目总览

### 一句话概念

> **一位以"绯红"为唯一颜色的少女，在褪色的"残响之境"中以剑斩开记忆，用三小时编织出一段关于"被忘却的人，值得被记起"的旅程。**

### 核心定位

| 维度 | 设定 |
|------|------|
| 类型 | 纯 2D 横版类银河城 + 动作探索 |
| 时长 | 主线 90-120 分钟 / 全收集 2.5-3 小时 |
| 美术 | 2D 手绘角色 + 分层水彩场景 |
| 主色调 | 淡灰 / 浅青 / 米白，**绯红为唯一高饱和色** |
| 受众 | 喜欢《空洞骑士》探索感 + 《GRIS》艺术性 + 《死亡细胞》手感的玩家 |

### 美术方向（关键卖点）

**核心视觉语言："少女是这个世界里唯一的颜色。"**

- **角色**：阿茜（Akane），动漫立绘风格，发色 / 瞳色 / 武器红丝为绯红，其余衣物素白
- **场景**：纯 2D 分层水彩笔触，前景 / 中景 / 远景以视差滚动制造空间；远景冷灰，近景柔和
- **UI**：极简白色细线 + 红色点缀，几乎"无 UI"的沉浸感
- **关键设计**：游戏中**所有红色都有意义**——重要物件、敌人弱点、可交互处、隐藏路径的痕迹，统一以"红"作为视觉指引。红 = "记忆未消逝"

### 世界观与故事

**「残响之境」**：被遗忘者的灵魂栖息地。当一个人在现实中被彻底忘记，TA 会在此地化作"残响"，逐渐褪色，直至消散。

**主角阿茜**：进入此地，寻找一位"再也想不起名字、却仍记得轮廓"的人。

**叙事方式**：
- 环境叙事为主，少且精致的对白
- 通过拾取「残响碎片」拼凑真相
- 多结局，由探索完成度决定（不是道德选择，是"记起多少"）

## 1. 项目总览

### 1.1 基础信息

| 项目 | 内容 |
|------|------|
| 游戏名 | 《茜响》（暂定，英文 *Akane's Echo*） |
| 类型 | 纯 2D 横版类银河城 / 动作探索 |
| 主平台 | **macOS（首发优先）** + Windows |
| 引擎建议 | Godot 4.6，纯 2D 节点与 GDScript，避免 3D 管线 |
| 目标时长 | 主线 90-120 分钟 / 全收集 2.5-3 小时 |
| 难度定位 | 硬核但公平，单一难度，通过存档点密度控制挫败感 |
| 输入支持 | 键鼠 + 手柄（Xbox / DualSense / 通用） |

### 1.2 AI 编程实现基线

- **使用 Godot 4.6 纯 2D**：CharacterBody2D、Area2D、TileMapLayer、AnimationPlayer、CanvasLayer 即可完成主体功能
- **不使用 3D、不使用复杂实时光照、不依赖自定义渲染管线**：视觉通过贴图、Shader 去色、粒子与视差层实现
- **房间制关卡**：每个房间是独立 `.tscn`，通过门节点切换，方便 AI 分批生成、调试和回滚
- **数据驱动但不过度抽象**：敌人、道具、房间门、碎片、存档点使用简单 Resource / Dictionary 配置，避免早期架构过重
- **可完成范围优先**：首发锁定约 46 个房间、48 个残响碎片、8 种普通敌人、4 种精英、4 个主线 Boss、1 个可选隐藏 Boss
- **先做手感原型**：第一阶段只验证移动、跳跃、攻击、影渡、残响槽、1 个敌人、1 个破壁点；这 7 项成立后再扩内容

### 1.3 设计三原则

> 开发期所有决策回归三原则判断：

1. **「红是唯一的颜色」**——美术、UI、关卡引导都围绕此核心
2. **「每个键都做两件事」**——攻击 → 蓄槽，资源 → 战斗 + 探索两用，能力 → 战斗 + 解谜两用
3. **「沉默胜过言语」**——文字精简至骨，靠场景说话

## 2. 操作方案

### 2.1 键位映射（PC 默认）

| 输入 | 键鼠 | 手柄 | 功能 |
|------|------|------|------|
| 移动 | A / D | 左摇杆 | 横向移动 |
| 跳跃 | K | B / ○ | 跳跃 / 贴墙时蹬墙 |
| 下落 / 读取 | S | 左摇杆下 | 下落平台 / 查看脚边残响 |
| 普攻 | J | X / □ | 三连斩 |
| 蓄力斩 | 长按 J | 长按 X / □ | 破护盾重击 |
| 冲刺斩 | A/D + J | 方向 + X / □ | 横向位移攻击 |
| 道具槽 1-4 | 1 / 2 / 3 / 4 | 十字键 上 / 右 / 下 / 左 | 触发装备的道具 |
| 残响 / 弹反 | 空格 | A / × | **核心键，详见 §4** |
| 互动 | E | Y / △ | 拾取 / 对话 / 残响阅读 |
| 地图 | M | 触摸板 / Select | 打开地图 |
| 装备整理 | TAB | Start | 打开背包 |

### 2.2 操作设计原则

- **左手只管移动 + 道具**（A/D/S + 1234），右手只管动作（J、K、空格）。键位分区明确，方便键盘与手柄双适配
- **空格作为最重要的键**，物理位置最大、最显眼，符合"它承担两个功能"的权重
- 手柄的 A 键（最大键）也对应空格——保持双输入习惯一致
- **K 键承担平台动作**：地面为跳跃，贴墙为蹬墙，长按影响跳跃高度；它的复杂度集中在移动手感，不和残响系统抢职责

## 3. 战斗系统详解

### 3.1 普攻（J 键）— 三连斩"风 · 雪 · 月"

```
第一段「风」    快速平斩    伤害 10    硬直 0.15s    连击窗 0.6s
第二段「雪」    上挑斩      伤害 12    硬直 0.20s    连击窗 0.7s
第三段「月」    旋身重击    伤害 25    硬直 0.45s    顿帧 6 帧
```

**手感细节：**
- 每段攻击包含 3 帧"启动" + 5 帧"判定" + 余下"收招"
- 第三段命中时镜头**轻震 + 时间停滞 4 帧**，营造打击感
- 红色粒子飞溅量与连击段数挂钩（一段少 / 三段爆）
- 攻击中可被"冲刺斩"或"弹反"打断，鼓励玩家随时反应
- 空中可使用第一段与第二段；第三段仅地面触发，避免空中连段失控

### 3.2 蓄力斩（长按 J）

- 蓄力时间：0.8 秒
- 蓄满后释放：伤害 50，必定破除"褪色护盾"
- 蓄力期间移动速度降低 50%，可被打断
- 视觉：剑身从白色逐渐渗出绯红，蓄满时整把剑燃起红芒

### 3.3 冲刺斩（A/D + J）

- 横向短距冲刺攻击，距离约 2 个角色身位
- 起步 6 帧无敌，命中敌人造成击退
- 冷却 1.5 秒，是主要的"机动 + 输出"工具
- 注意：与"影渡（前冲）"是不同动作，影渡冷却短无伤害，冲刺斩冷却长有伤害

### 3.4 受击与生命

| 项目 | 数值 |
|------|------|
| 初始 HP | 100 |
| 初始残响槽上限 | 100 |
| 受击硬直 | 0.4 秒 |
| 受击无敌时间 | 0.8 秒 |
| 死亡惩罚 | 回到上一个"残响碑"，掉落 50% 绯红丝缕（可回原地拾回，《空洞骑士》式） |

## 4. 残响系统（核心创新）

> 这是本作的招牌系统。**空格键一个键，承载战斗 / 探索 / 防御三重作用**。这个系统好不好，决定整个游戏好不好玩。

### 4.1 残响值的获取与消耗

```
【积攒】
  普攻命中             +5 / 击
  冲刺斩命中           +8 / 击
  蓄力斩命中           +20 / 击
  弹反成功             +30（后期通过技能解锁这一项）
  受到伤害             -10（受伤会扣槽，鼓励玩家不要被打）

【消耗】
  残响态（满 100%）    战斗增幅，6 秒
  破壁（≥50%）         探索用，劈开褪色之壁
  弹反（无消耗）        前期：仅前期版本无消耗弹反；后期觉醒后变为"凝时斩"
```

### 4.2 三种用途详解

#### A. 残响态（消耗 100%）
- 持续 6 秒
- 时间减速 0.7 倍（敌人慢，主角不慢）
- 攻击伤害 +50%，附绯红拖影特效
- 命中时回血（每击 +2 HP）
- 解除受击硬直
- **设计意图**：Boss 战的爆发期 / 危险时刻的紧急按钮

#### B. 破壁（消耗 50%）
- 对世界中的"褪色之壁"使用，劈开它
- 这是某些区域 / 捷径 / 隐藏房的**唯一**进入方式
- **设计意图**：制造"该攒着打 Boss 还是花掉去探索"的纠结

#### C. 弹反凝时斩（详见 §4.3）— 后期觉醒能力

### 4.3 ⚔ 弹反凝时斩（重点设计）

> 这是阿茜的**第四个也是最后一个**探索能力，玩家在游戏后段（约 75% 进度）通关白栀庭后获得。设计目标：让玩家"突然变强"，回到旧地图爽快通行。纯 2D 版本中，它主要解决"敌群与机关节奏"而不是复杂空间寻路。

**前置条件**：
- 解锁前空格键 = 残响态 / 破壁
- 解锁后空格键多了一个判定层：**敌人攻击命中前 0.4 秒按下空格 → 进入弹反流程**
- 平时按空格仍正常使用残响态 / 破壁，两套机制不冲突

**弹反执行流程**：

```
[第 1 步] 敌人攻击即将命中，玩家按下空格
            ↓
[第 2 步] 阿茜挑剑挡格，绯红火花迸射
            ↓
[第 3 步] 全屏时间凝固 1.5 秒（仅主角能动）
            画面去色至灰白，仅红色仍在
            ↓
[第 4 步] 阿茜自动锁定当前屏幕内最多 5 个敌人
            进行 8 次跳跃斩击
            （每个敌人至少被打 1 下，剩余打击分配给威胁最高目标）
            ↓
[第 5 步] 时停结束，画面色彩回归
            斩击伤害一次性结算
            残响槽 +30
```

**数值定位（B 档：中风险中回报）**：

| 项目 | 数值 |
|------|------|
| 判定窗口 | 0.4 秒 |
| 弹反失败后果 | 正常受伤 + 0.5 秒额外硬直（小惩罚） |
| 单次斩击伤害 | 18（合计最多 144 单目标） |
| 冷却时间 | 4 秒（防止刷判定） |
| 残响值消耗 | 0（凝时斩不消耗，但成功后回 30，鼓励连续攻击） |
| 适用敌人 | 一切非 Boss 敌人。Boss 可弹反但不触发全屏连斩，仅破除一次攻击并制造 1 秒输出窗 |

**为什么是后期解锁**：
1. 战斗终极目标从"贴脸输出"变为"读招应对"，体验有进化感
2. 回到前期区域时，原来需要硬碰硬的小怪群一招清场，**爽快感自然产生**
3. 配合二周目，玩家可以挑战 100% 弹反通关，提升复玩价值

## 5. 道具系统

### 5.1 道具栏机制

- 玩家有 4 个固定槽位（1 / 2 / 3 / 4 键）
- 在"残响碑"前可自由更换装填的道具
- 全游戏共 **12 种道具**，玩家需要根据 Boss / 区域特性选择 Build

### 5.2 道具一览（v1）

| 道具名 | 类型 | 槽位用法 | 效果 |
|--------|------|----------|------|
| 红茶 | 消耗品 | 按下使用 | 回复 40 HP，3 个充能，残响碑前补满 |
| 樱酒 | 消耗品 | 按下使用 | 回复 50 残响值，2 个充能 |
| 封符 | 投掷物 | 按下投出 | 中距离命中后破甲，敌人 5 秒受伤 +30% |
| 咒纸 | 投掷物 | 按下投出 | 直线穿透，对褪色系敌人额外伤害 |
| 地刺符 | 陷阱 | 按下放置 | 在脚下放置，5 秒内踩到的敌人受伤 |
| 迷雾符 | 陷阱 | 按下放置 | 制造 4m 烟雾，敌人在内看不见你 |
| 残响碎瓷 | 即时 | 按下触发 | 立刻 +30 残响值，1 个充能（稀有补给） |
| 风铃护符 | 被动 | 装备即生效 | 受到伤害时 30% 概率不扣残响值 |
| 朱玉 | 被动 | 装备即生效 | 残响态时间 +2 秒 |
| 旧怀表 | 被动 | 装备即生效 | 弹反判定窗口 +0.1 秒（解锁弹反后必备） |
| 缠布 | 被动 | 装备即生效 | 死亡时丢失的丝缕减少 50% |
| 紫阳铃 | 被动 | 装备即生效 | 靠近隐藏物时铃铛轻响（探索辅助） |

### 5.3 设计意图

- **6 个主动 + 6 个被动**，玩家通常会装 1-2 主动 + 2-3 被动
- 投掷 / 陷阱在 Boss 战发挥大作用，被动符在探索期发挥大作用
- 没有"全能道具"，每次进入新区域都值得重新搭配 → 制造决策乐趣

## 9. 主角阿茜的人设细化

### 9.1 视觉

- 16-17 岁少女，身高约 158cm，身材瘦削
- 银白短发（齐耳偏短），左侧别一支绯红丝带
- 瞳色：右眼银灰，**左眼绯红**（异色瞳，与"凝时斩"觉醒呼应）
- 服装：白色高领无袖短上衣 + 黑灰百褶短裙 + 白色及膝靴 + 红色细腰带
- 武器：单手细身长剑，剑柄缠红绳，剑刃微泛白

### 9.2 性格反差

**外**：战斗与探索时，沉稳、冷静、表情几乎不变。面对 Boss 也只是平静地举剑。
**内**：内心独白却格外抽象——会突然吐槽场景里的怪东西、给敌人乱起外号、对自己的窘境冷嘲热讽。

**内心独白触发场景**：
- 进入新区域时（约 1-2 句）
- 残响碑小睡时（与世界 / 自己对话）
- 击败 Boss 后（情感峰值）
- 触发某些场景细节时（探索奖励）

**独白示例**：

> 进入燃尽书馆：
> 「书是好东西，被烧的书也是好东西，这是我能给烧书的人最克制的评价。」

> 残响碑小睡：
> 「这块石头比家里的床舒服。这件事让我感到忧伤。」

> 击败第二位残响：
> 「……抱歉，我不知道你的名字。
> 但我会记住你最后那个动作。
> 那个动作很丑，但是很认真。」

> 看到一棵长歪的树：
> 「这棵树长得像我室友。我没有室友。」

### 9.3 NPC：渡舟人「咏」（贯穿始终的固定 NPC）

- 中央枢纽"忘川渡口"的渡船人，性别中性，看不清面容
- 不主动给任务，只在阿茜每次回到枢纽时随便聊几句
- 对话不推进剧情，但每次都会**留下一句意味深长的话**
- 是玩家唯一稳定的"声音"，制造温暖的对照

# 第二卷：世界观与剧情

> 📌 本卷定位：建立游戏的世界设定、核心隐喻、主线剧情、四位 Boss 的故事弧、多结局分支与文本量预算。剧情服务于"环境叙事 + 少而精对白"的呈现方针。

---

## 1. 世界观总纲

### 1.1 世界结构「三界」

```
现实界（活人的世界）
    │
    │  当一个人逐渐被忘记 ──→ 灵魂滑落
    │
残响之境（本作舞台）
    │
    │  灵魂在此栖息，依靠"被记得"维持形态
    │  完全被忘记 ──→ 褪色
    │
忘川（消散之地）
    褪色的灵魂归于虚无
```

### 1.2 残响之境的运作法则

- 这里的居民称为「残响」——TA 们不是死人，而是**仍被遗忘的人**
- 每个残响保留**别人对 TA 印象最深刻的一瞬**作为形态
- 当现实界还有人记得 TA，TA 就保有色彩；记忆模糊，TA 开始褪色；彻底忘记，TA 化为白
- **完全褪色的残响**会失去自我，成为游荡的"白怨"——本作的普通敌人
- 极少数残响**抗拒褪色**，凝结成"未送达的残响"——本作的 Boss

### 1.3 「红」的隐喻

> 红 = 还活着的记忆，没有被忘记的证明。

- 阿茜全身唯一的颜色 → 她身上承载着某段强烈记忆
- 重要物件呈红色 → 这是某个残响留下的"未褪色之物"
- 凝时斩时画面去色仅留红 → 强迫世界承认"这一刻仍被记得"

### 1.4 时间法则

残响之境**没有时间**。
- 永远是同一个黄昏
- 阴影不会移动
- 但水会流，雨会下，钟会响——这些是"被记得的瞬间"，所以仍在动

---

## 2. 主角设定

### 2.1 阿茜的真实身份（核心谜底）

> ⚠️ 这是游戏的最大谜底，玩家在第 4 区域才会逐渐意识到。

**阿茜本人，也是一名残响。**

- 她在第一幕醒来时不记得自己为什么来这里，只记得"要去找一个人"
- 她身上保留色彩 = 现实中**仍有一人记得她**
- 她的左眼绯红 = 那个人对她最深的印象是"血色的左眼"
- 她最终要找的"那个人"——其实就是**那个还在记得她的人**

### 2.2 阿茜与"那个人"的关系（剧情驱动力）

**剧情真相**：阿茜生前是一名年轻的剑术老师，"那个人"是她唯一的学生——一个怯懦的、连握剑都不敢的少年。

她在保护少年时死去。少年活了下来，从此每天都在想她，所以她虽死，未褪。

但是——
- **少年正在长大，记忆开始模糊**
- 阿茜在残响之境每走一步，少年的记忆都在现实中褪色一分
- **她越接近真相，自己就越接近消散**

### 2.3 阿茜的反差人设落地

阿茜的"冷面 + 抽象内心"在剧情上的解释：
- 她生前就是这样的人——表面是严师，内心其实在吐槽一切
- 这种性格**正是少年最深刻记住她的部分**
- 所以她在残响之境保留下来的形态，就是这个反差感本身

> 🎯 设计妙处：玩家以为阿茜的性格是单纯的人设趣味，但通关后会意识到——**她抽象的吐槽，是少年记忆中最不愿失去的东西**。

## 3. NPC「咏」的设定

### 3.1 表面身份

- 忘川渡口的渡船人，沉默温和
- 不推剧情，只闲聊
- 玩家每回到枢纽都会与之交谈一两句

### 3.2 真实身份（隐藏剧情）

> 通过收集 36 + 残响碎片可解锁咏的真相。

**咏，是阿茜自己的"遗忘"具象化。**

- 阿茜每去一个区域，都会丢掉一部分自己（被少年遗忘的部分）
- 这些被丢掉的部分，凝结成了"咏"
- 咏看着阿茜一步步走向终结，无法阻止，只能在每次相遇时留下一句话

### 3.3 咏的对白示例

每次回到枢纽，咏只说一句话，不重复：

> 「今天的雨，下得比昨天的轻。」
> 「你最近，越来越像一个真正的人了。」
> 「我曾经也想过去找谁，后来我忘了那是谁。」
> 「红色这东西，不该用得太省。」
> 「下一站会很难。但你已经过了上一站，对吧。」
> 「如果有一天我不在这里了，记得自己解开渡船。」

最后一句话，会在玩家通关前最后一次回到枢纽时出现。

### 4.1 五幕结构

```
【第二幕】燃尽书馆 ─────────── 35-60 min   主题：后悔
```

## 5. 四位 Boss 的故事

> 设计哲学：每位 Boss 都是一个具象化的"被遗忘者"，战斗中喃喃自语透露生前。**击败不是杀死，而是送行**。

### 5.1 Boss 1「雨衣的男人」 — 主题：离别

**外形**：穿着褪色雨衣的中年男人，看不见脸，雨水从领口不断渗出。手中握半把折断的伞。

**生前**：一位送女儿上学的父亲。某天送完伞后再也没回家。

**战斗中喃语**：
- 「记得带伞……记得带伞……」
- 「车站三号口，别走错。」
- 「她……她现在多大了。」

**击败演出**：
雨衣脱落，露出空空的内里。一把完整的红色小伞从空中飘落，缓缓落进他原来的位置。

**阿茜独白**：
> 「这位父亲。
> 您的伞我会带给她。
> 但是您的话——
> 您的话她已经不需要了。她长大了。」

**奖励**：道具「红茶」蓝图 + 缠红丝技能（叙事置换：缠红丝其实是这位父亲的领带变的）

### 5.2 Boss 2「未写完的人」 — 主题：后悔

**外形**：上半身是穿白衬衫的青年，下半身是飘散的纸页。一只手握着永远蘸不到墨水的笔。

**生前**：一位作家，写了一辈子的小说没完成就去世了。

**战斗中喃语**：
- 「下一句、下一句应该……」
- 「主角不该死的。」
- 「再给我十分钟。再给我十分钟。」

**特殊机制**：
战斗中场景的书页会飘起，玩家被纸页击中会掉落"未写完的字"，需要捡回（机制象征"把他的话捡回来"）

**击败演出**：
他终于停笔。所有飘散的纸页聚拢成一本书，然后燃成红色灰烬。

**阿茜独白**：
> 「写完了。
> 您写完了。
> 哪怕只是停笔，也算是写完了。」

**奖励**：静音步技能 + 道具「咒纸」

### 5.3 Boss 3「灯塔守」 — 主题：孤独

**外形**：穿厚重大衣的老妇人，提着一盏不熄的红灯。脸上裹着围巾。

**生前**：海边灯塔的守塔人，一辈子没有结婚，等一艘永远没回来的船。

**战斗中喃语**：
- 「船今天会回来。」
- 「船今天，也许会回来。」
- 「船……」

**特殊机制**：
她周围有红灯光圈，圈内玩家行动减速。但是——**只有红光圈内能看见她的弱点**。强迫玩家走进她的孤独。

**击败演出**：
灯熄了。海面上远远的，一艘小船的剪影出现，然后又消散。

**阿茜独白**：
> 「船没有回来。
> 但是您来过了，这里。
> 我觉得这件事比船重要。
> ……虽然您不一定同意。」

**奖励**：影渡升级（距离 +50%）+ 道具「风铃护符」

### 5.4 Boss 4「持剑的少女」 — 主题：执念 / 真相

**外形**：白发，左眼绯红，握细身长剑——**和阿茜一模一样**。

**生前**：阿茜本人，过去那个"决意保护学生而死"的瞬间。

**战斗中喃语**：
- 「保护他。」
- 「把剑握紧。」
- 「不要回头。」
- （战斗后期）「……你是谁。」
- （濒死）「啊。是我。」

**特殊机制**：
- 这是全游戏最难的战斗
- 她会使用阿茜目前所有的招式（包括弹反凝时斩——主人会在战斗中第一次见到这招的样子）
- 击败她的唯一方法：**学会她的弹反**——这就是凝时斩的获得方式

**击败演出**：
她跪下，剑插入地面。她抬头看向阿茜，露出第一次微笑。
然后她身上的红色——左眼、剑柄红绳、腰带——**飘起来，融入阿茜身上**。
阿茜的红色变深了。她终于完整了。
但同时，她身体的边缘开始**轻微发白**。

**阿茜独白**：
> 「啊。
> ……
> 是我啊。」

> （沉默五秒）

> 「我以为我会哭。
> 但其实没什么好哭的。
> 一个人最难原谅的就是自己——
> 现在我把自己原谅了，剩下的就简单多了。」

> 「最后一站了。
> 走吧。」

**奖励**：凝时斩 + 静默之心 ×1（最后一颗）

## 6. 多结局设计

### 6.1 结局触发机制

通关前，阿茜回到忘川渡口。**咏不见了**。渡船停在岸边。

阿茜可以选择：
- 上船 → 渡船向忘川驶去
- 不上船 → 回到残响之境

但**真正决定结局的不是这个选择**——而是玩家在游戏中**收集了多少残响碎片**。

### 6.2 三种结局

#### 结局 A：「消散」（残响碎片 < 24 / 48）
- 阿茜上船，渡船驶入白雾
- 她身上的红色一点点褪去，最后只剩剑柄那一抹
- 字幕：「她终于被忘记了。」
- 现实界画面：少年长大成人，结婚那天早晨，他对着镜子愣了一下——**好像有什么忘了**——然后摇摇头，出门了。

#### 结局 B：「驻留」（残响碎片 24-39 / 48）
- 阿茜不上船，转身回到残响之境
- 她成为新的渡船人，等下一个迷路的人
- 字幕：「她选择留在被忘记的人身边。」
- 现实界画面：少年长大成人，每年在固定那天会去某个山坡，放一把红伞。**他不知道自己为什么这么做。**

#### 结局 C：「归还」（残响碎片 40+ / 48 + 击败隐藏 Boss「最初的咏」）

> 隐藏 Boss「最初的咏」：收集 4 片高光残响碎片并击败白栀庭 Boss 后，在忘川渡口开启隐藏入口。

- 阿茜走上船，但没有渡向忘川——她让船**逆流**，驶向现实
- 她变成一缕红光，飞回少年身边
- 字幕：「她回去了。回到那个仍在记得她的人身边。」
- 现实界画面：少年（已是青年）站在自己开的剑道馆前，门口风铃响起。他抬头，看见门口飘来一片红色花瓣。**他笑了，眼眶有泪。**他对着空气说：
  > 「老师，您回来了。」

### 6.3 结局成就与新游戏 +

- 完成任一结局后开启**新游戏 +**
- 新游戏 + 中咏的对白完全不同（她知道阿茜回来过）
- 二周目可挑战「无伤通关」「100% 弹反通关」获得专属皮肤（白色阿茜 / 红色阿茜）

## 8. 残响碎片设计

### 8.1 形式

- 散落在地图各处的红色小物件（耳坠 / 信笺 / 玩具 / 戒指 / 干花）
- 拾取后弹出 1-3 句话，类似拼图的一片
- 不强制阅读，但不阅读会错过一个层次的故事

### 8.2 残响碎片示例

> **「红色发圈」**（拾取于濡羽之森）
> 「她说红色不衬她。她又把它戴上了。」

> **「半张照片」**（拾取于燃尽书馆）
> 「照片只剩一半。剩下的一半，是被人小心地剪走的——而不是撕的。」

> **「钢琴键」**（拾取于沉钟海）
> 「这架钢琴只剩中央 C 还能响。
> 她每天都按一下。」

> **「红色蜡笔」**（拾取于白栀庭）
> 「这是少年画给老师的画。
> 他画得很丑。
> 老师珍藏了一辈子。」

### 8.3 碎片分布

- 第 1 区域：9 片
- 第 2 区域：10 片
- 第 3 区域：10 片
- 第 4 区域：13 片
- 中央枢纽：6 片

**关键碎片**：每区有 2-3 片是"高光碎片"，集齐 4 片"高光"才能解锁结局 C 的隐藏祭坛。

# 第三卷：关卡设计与流程

> 📌 本卷定位：将世界观与系统落地为可玩的物理空间。包含整体地图结构、四区域 + 中央枢纽的详细房间设计、锁钥分布、回溯路径、节奏控制与玩家引导。

---

## 1. 地图设计哲学

### 1.1 三大原则

**原则一：每个房间都要有"被记住的理由"**
玩家通关后回想，每个房间应该都能说出"那个房间是干嘛的"——而不是"那是个走廊"。即使是过渡空间，也要有一个**视觉锚点**或**叙事钩子**。

**原则二：迷路是设计，不是事故**
玩家会迷路，但不能"卡住"。每次迷路时都应该至少**误打误撞撞见点东西**——一片碎片、一个新敌人、一段奇怪的环境对白。**迷路 ≠ 浪费时间**。

**原则三：回溯要奖励，不要惩罚**
后期回旧区域时，原来的小怪不该让人烦躁。所以——
- 回溯时，已清理的小怪**不会刷新**（除非死亡 / 大休息）
- 回溯路径有"快速通道"（破壁打开的捷径）
- 回溯目标要**视觉上明显**（远远就能看到"那里有红色，去那里"）

### 1.2 整体地图拓扑

```

                            │   白栀庭（4 区）   │
                            │     执念 / 真相    │

                                       │

                                       │ 才能开启的门
                                       │
        ┌──────────────────────────────┴──────────────────────────────┐
        │                                                             │
        │                  忘川渡口（中央枢纽）                          │
        │                  · 残响碑 · 绯红铸炉 · 咏 ·                    │
        │                                                             │
        └──┬──────────────────┬─────────────────────┬───────────────┬─┘
           │                  │                     │               │
```

- 4 大区域 + 1 中央枢纽 = 类银河城的"轮辐"结构

- 白栀庭独立在最上方，需要前三区都通关才能进入
- 隐藏入口连接结局 C 的祭坛

## 2. 中央枢纽：忘川渡口

### 2.1 区域基调

- 永远的黄昏，紫灰色的雾
- 一条静止的河（忘川），河面映出主角的倒影但**不映出咏**
- 一艘渡船永远停在岸边，船身有红色的纹路
- 一座低矮的木屋（咏的小屋）
- 一个燃着红炭的铁炉（绯红铸炉）

### 2.2 房间布局（5 间）

```
[1] 醒来之屋     ──→  阿茜苏醒的小木屋（仅序幕使用）
[2] 渡口码头     ──→  咏所在地，对话点
[3] 绯红铸炉     ──→  武器升级
[4] 残响石阵     ──→  存档点 + 4 个区域传送门
[5] 河岸小径     ──→  区域入口 + 终幕渡船；高光碎片集齐后显现隐藏入口
```

### 2.3 设计要点

- **每次回到枢纽，咏的位置 / 姿势会微变**（坐着 → 站着 → 看河 → 看船）
- 河面上漂浮的"未送达的信"会随着剧情进度增多
- 第 4 幕开始后，铸炉的火焰会变成纯白色
- 终幕时，咏的小屋空了，门虚掩着

## 3. 第 1 区域：濡羽之森

### 3.1 区域档案

| 项目 | 内容 |
|------|------|

| 视觉 | 永夜的雨林，黑色树干，水汽弥漫，地面湿润反光 |
| 主色 | 深绿 / 墨黑 / 米白 |
| 红色焦点 | 散落的红色伞骨、雨衣男人的红伞 |
| 房间数 | 9 |
| 残响碑 | 2 |
| 残响碎片 | 9 |
| Boss | 雨衣的男人 |
| 解锁能力 | 影渡（区域中段获得，作为教学）|
| 区域时长 | 20-25 分钟 |

### 3.2 房间序列

```
[F-01] 林缘渡桥        横向移动 + 跳跃教学
[F-02] 滴水洞口        敌人初见（白怨小型）
[F-03] 倒木裂沟        ★ 影渡获得点（教学房）
[F-04] 雨幕岔口        双向分支（区域小枢纽）

[F-08] 父亲的足迹      隐藏路径起点 + 残响碑 2
[F-09] 雨衣下的祭坛    ★ Boss 战，胜利后获得缠红丝
```

### 3.3 关键房间细节

#### F-03「倒木裂沟」— 影渡教学房
- 一道宽 4m 的裂缝挡路
- 第一次进来时无法过去，但裂缝对面**能看到**一片红色（视觉钩子）
- 房间里有一只发着微红光的"残响兔"，触摸后获得影渡
- 获得后强制使用一次（教学）穿过裂缝
- 对面拾起的不是道具——而是一段独白触发：
  > 「跨过去之后才发现，对面的风景，和这边其实差不多。
  > 这件事让我感到欣慰。」

#### F-04「雨幕岔口」— 三叉枢纽
- 整个区域的中枢，每条路都能看见红色提示物（北：红色羽毛 / 东：红色伞骨 / 南：红色脚印）
- 玩家可自由选择探索顺序
- **设计巧思**：两条路最终都会汇回 F-09，所以不会"走错路"

#### F-07「折伞之地」— 残响碎片密集区
- 整面墙嵌着无数把折断的伞
- 3 片残响碎片集中于此（区域内最高密度）
- 唯一一个能听见现实界声音的地方——隐隐有小女孩在远处喊"爸爸——"
- 阿茜独白：
  > 「这些伞，怎么可以在同一个地方折断这么多次。
  > 啊，原来不是同一个地方。
  > 是同一段记忆。」

#### F-08「父亲的足迹」— 隐藏路径起点
- 看似死路，但地面有一串红色的湿脚印
- 跟着脚印走到墙边——一面褪色之壁
- **此时玩家无法破壁**（残响值还不够 / 没领悟破壁机制），需要后期回来
- 后期破壁后开启隐藏房 F-08b：拾取高光残响碎片 + 强力被动符
- **教导玩家：墙壁是可以打破的，但不是现在**

#### F-09「雨衣下的祭坛」— Boss 战房
- 中央一根孤独的电线杆，挂着一盏摇晃的红灯笼
- 雨势加大，整个屏幕被雨幕覆盖
- Boss 出场前先看到一把伞从天上飘落
- 战斗音乐：低音提琴 + 雨声 + 渐渐加入的小女孩童谣（不寒而栗的温柔）

### 3.4 区域回溯节点

获得后期能力后回到本区域可解锁：
- **缠红丝**：F-05 鸦栖之木顶部的鸟巢，藏 1 残响碎片
- **静音步**：F-07 折伞之地的碎枝平台，可无声通过，藏 1 残响碎片
- **凝时斩**：F-08b 隐藏房的高速机关，暂停后获得"高光碎片 1/4"

```
[L-01] 书馆前厅          残响碑 1，缠红丝节点教学
[L-02] 长梯回廊          横版垂直探索，使用缠红丝上拉
[L-03] 阅读室            分支：书架推动谜题 + 普通战斗

[L-09] 「END.」          ★ Boss 战，胜利后获得静音步
```

### 4.3 关键房间细节

#### L-04「旋转书塔」— 核心解谜房
- 一座 3 层书塔，每层都在缓慢旋转（每层方向不同）
- 玩家需要在合适的时机踏上 / 跳离每层
- 顶端有一颗**未燃尽的火星**——拾取后获得"残响碎瓷"道具蓝图
- **机制目的**：教玩家"等待时机"，是为后期弹反凝时斩做的节奏铺垫

#### L-05「半烧的回忆室」— 高光残响碎片
- 房间地面是一张巨大的、未完成的书稿
- 玩家走过时，脚印会在书稿上留下红色痕迹，但稿纸越燃越多
- 5 秒内必须找到房间中央的红色书签触摸，否则……什么都不会发生（叙事性失败）
- 触摸书签后弹出独白：
  > 「我替您把它读完了。
  > ……虽然您写得真的，
  > 一般。」

#### L-06「镜廊」— 重要叙事伏笔
- 走廊两侧是无数面破碎的镜子
- 镜中倒影**不是阿茜**，而是一个戴眼镜、穿白衬衫的青年（少年长大后的样子）
- 阿茜路过时镜中人会跟着走，偶尔停下看她
- 阿茜独白：
  > 「这位先生，您看错人了。
  > ……还是说，是我看错了。」

> 🎯 这是为第 4 区域的"真相揭示"埋的伏笔。一周目玩家会觉得诡异，二周目会泪目。

#### L-08「未完成的章节」— Boss 战前室
- 一张巨大的桌子，一支永远蘸不到墨水的笔，一摞稿纸
- 稿纸上写着 Boss 生前最后一段文字（由玩家选择 1 张读 / 不读）
- 读了之后，Boss 战中他的喃语会有一句额外的台词

#### L-09「END.」— Boss 战房
- 一片纯白虚空，地面是写着"END."的稿纸
- 战斗中地面会被红色墨水逐渐覆盖
- Boss 的笔尖永远悬在 END. 上方，写不下那个句号

### 4.4 区域特色机制：飘浮书页

- 燃尽书馆中部分房间有飘动的书页
- 书页击中玩家会掉落"未写完的字"
- 玩家需要在 10 秒内捡回，否则**这一处的残响碑会暂时失效**（叙事性惩罚）
- **这是全游戏唯一的"非战斗惩罚机制"**，制造紧迫感

```
[S-03] 海星墓园             解谜：星图对应（残响碎片密集）
[S-04] 钟摆回廊             ★ 节奏型平台房
[S-05] 沉船骨架             隐藏路径起点 + 敌群战
[S-06] 钟舌缝隙             ★ 需要影渡穿过的能力检验
[S-07] 旧船长室             高光残响碎片 + 残响碑 2
[S-08] 灯塔顶层             ★ Boss 战前室
[S-09] 红灯之处             ★ Boss 战，胜利后影渡距离 +50%
```

### 5.3 关键房间细节

#### S-03「海星墓园」— 解谜房
- 海床上散落着无数海星化石
- 中央有一座旧航海仪，玩家需根据天上的星座（关卡背景）旋转仪器对应海星
- 解开后地面浮起一道暗道，进入隐藏小房间获取静默之心
- 阿茜独白：
  > 「我从来没学过看星星。
  > 但是被搁浅了之后，
  > 总要找点事做。」

#### S-04「钟摆回廊」— 节奏型房间
- 一条横版长廊，5 个巨大的钟摆来回摆动
- 玩家需要在钟摆缝隙间穿行，命中会受到重伤并被弹回最近安全平台
- 每个钟摆节奏不同
- **设计目的**：训练玩家"读节奏 + 预判 + 等待时机"，是为弹反做铺垫的玩家技巧训练
- 中段有一个安全休息平台，平台上有一片残响碎片：
  > 「她每天都听着钟声入睡。
  > 直到有一天，钟停了。
  > 她也再没醒来。」

#### S-06「钟舌缝隙」— 锁钥瓶颈
- 一道极窄的缝隙，只能用影渡的无敌帧穿过
- 这是**强制性的能力检验**——证明玩家已掌握影渡
- 通过后是直通 Boss 战的捷径

#### S-07「旧船长室」— 叙事房
- 一间保存完好的船长室（与外面的废墟形成强烈对比）
- 桌上摆着两副餐具，但只有一个座椅
- 床上枕头有两个，但只有一边凹陷
- **房间里没有敌人，没有谜题，只有探索**——这是给玩家"喘息"的房间
- 阿茜独白：
  > 「她吃了一辈子饭，桌子对面都摆着碗。
  > 这件事我没办法评价。
  > 我只能记住。」

#### S-09「红灯之处」— Boss 战房
- 灯塔顶层，巨大的红灯居中悬挂
- 战斗时灯光会**忽明忽暗**——光圈内行动减速但能看见 Boss 弱点；光圈外行动正常但 Boss 进入隐身
- 玩家必须**主动走入她的孤独**才能赢
- 战斗音乐：单簧管独奏 + 远处的钟声

### 5.4 区域特色机制：灯光交替

- 沉钟海多个房间有"红灯效应"——红光圈内是减速 + 可见的领域
- 后期获得静音步后，可在红光圈外完全隐身
- 这个机制将与第 4 区域的"白色之光"形成镜像对照

### 6.3 关键房间细节

#### H-02「白栀长廊」— 强叙事开场
- 一条全白的长廊，无敌人无谜题
- 阿茜每走 5m 触发一次独白，共 4 段：
  > 「这里太干净了。」
  > 「干净得像没有人来过。」
  > 「干净得像……我自己。」
  > 「……这种话不该是我说的。」

#### H-03 / H-05 / H-07 / H-09「四个回忆庭」— 区域核心机制
- 4 个独立的小庭院，每个庭院里有一段"回放"——白色剪影演绎一段过去
- 玩家可以**走进剪影中**，剪影会被打散，掉落"高光残响碎片"
- 但走进剪影会触发战斗——和**白色的过去自己**对决
- 4 场战斗难度递增，是 Boss 战的"试炼"

**4 个回忆的内容**：
1. H-03：阿茜年幼时第一次握剑（少女剪影 vs 阿茜）
2. H-06：阿茜成为剑术老师，第一次教学（年轻女子剪影）
3. H-09：阿茜遇见学生，那个怯懦的少年（双人剪影，阿茜只能攻击其中一个）
4. H-09：阿茜保护学生，刺杀来袭者（**剪影中阿茜中剑倒下** —— 这是她死亡的瞬间）

> ⚠️ H-09 战斗后，玩家终于看清——白色剪影中有一个倒下的、和自己长得一样的人。
> 这是真相揭示前的最后一击。

#### H-06「时间静止之屋」— 重要叙事房
- 一间小教室，桌椅整齐，黑板上写着字（被风干的粉笔）
- 桌上放着一本剑术教材，翻开的页面上有阿茜年轻的字迹
- **没有敌人，没有谜题**——纯叙事
- 玩家可以坐在教师讲台后（按 E）
- 坐下后阿茜独白：
  > 「我以前是不是站在这里？
  > 还是坐在这里。
  > 这个椅子比我家那块石头还硬。」
  > （沉默）
  > 「……怎么我会知道哪个椅子硬。」

> 🎯 这是真相伏笔的最大爆点之一——阿茜第一次"想起"了什么。

#### H-08「学生的房间」— 真相揭示房
- 一间少年的卧室，墙上挂着许多画
- 画上都是同一个戴红丝带的女老师，从一张到几十张
- 最大的一张画——是阿茜的肖像，下面用孩子的字写着「老师」
- 阿茜走到画前会停住很久，没有独白
- 直到玩家按 E 互动：
  > 「……」
  > 「这画得真丑。」
  > 「但是画得很认真。」
  > （长时间沉默）
  > 「我想起来了。」
  > 「全部。」

> 这一刻是游戏情感最高点之一。**没有 BGM，只有钟摆声**。

#### H-10「持剑的少女」— Boss 战房
- 一片纯白空间中央，唯一一朵盛开的红栀子
- Boss 站在花前
- 战斗中，每当阿茜命中她，红色就从她身上飘到阿茜身上
- 战斗结束时，红栀子凋零，花瓣飞起化作绯红光融入阿茜
- 凝时斩在此战首次以"对手使用"的形式出现
- 玩家必须**学会她的招式（弹反）才能击败她**

### 6.4 锁钥提示

H-09 之后，主区域有 2 处需要凝时斩才能进入的隐藏区域。这些区域各提供 1 片残响碎片，并把玩家引回中央枢纽的隐藏入口。

## 10. 引导设计

### 10.1 玩家引导三层

**第一层：视觉引导（最强）**
- 红色 = 该去的方向 / 重要物件
- 视野中永远至少能看到一处红色（除非在叙事性"无方向"房间）

**第二层：声音引导**
- 静默之心方向有微弱钟声
- 残响碎片附近有铃铛响
- Boss 房入口有低音持续震动

**第三层：地图引导（最弱）**
- 地图只标已探索房间
- 残响碑可在地图上传送（解锁后）
- **故意不标残响碎片位置** —— 鼓励真探索

### 10.2 防卡死机制

- 残响碑前可"咏唱"——简短的方向暗示（咏的声音从远处传来）
- 主菜单有"提示"按钮，每 30 分钟可使用一次，会显示一个未解锁的残响碎片大致方向
- 但**不显示主线下一步**——硬核难度，剧情进度全靠玩家自己摸

# 第四卷：敌人与 Boss 详细数据

> 📌 本卷定位：定义全游戏所有战斗单位的数值、行为模式、AI 决策逻辑、Boss 多阶段设计与战斗节奏。本卷数据为 v1.0 调校基线，正式开发期需根据手感测试微调。

---

## 1. 敌人设计哲学

### 1.1 三大原则

**原则一：每只敌人都要"教点东西"**
没有"为了让玩家受伤而存在"的敌人。每只敌人的招式都对应一种玩家技能的训练（移动、节奏、反应、记招）。

**原则二：白怨 = 残响的影子**
所有普通敌人都是"完全褪色的残响"，所以全部呈白色。它们生前是普通人，死后无人记起。**击杀普通敌人也带有微弱的悲伤感**。

**原则三：数量恰到好处**
首发锁定 **8 种普通敌人 + 4 种精英敌人 + 4 个 Boss + 1 隐藏 Boss = 17 个战斗单位**。每个区域 2 种普通敌人足够形成组合变化；额外普通敌人作为 1.1 版本备选，不阻塞首发。

### 1.2 敌人分类

| 大类 | 数量 | 用途 |
|------|------|------|
| 普通敌人（白怨） | 8 种首发 + 4 种备选 | 探索期主力，每区 2 种首发敌人 |
| 精英敌人（褪色未尽者） | 4 种 | 每区一种，是该区域的小考验 |
| Boss（未送达的残响） | 4 个 | 区域终点 |
| 隐藏 Boss | 1 个 | 隐藏区域终点 |

### 1.3 敌人共通规则

- 所有敌人都是白色基底 + **极小的红色印记**（眼睛 / 心脏位置 / 武器尖端），印记 = 弱点
- 弱点位置受到攻击额外伤害 +30%
- 所有敌人会在死亡前有一个"白光定格 0.3 秒"——给玩家最后一次取消攻击的机会，避免误伤已死敌人
- 所有敌人**不会复活**（除非玩家死亡 / 在残响碑大休息）

## 4. Boss 1：雨衣的男人（详细战斗设计）

### 4.1 战斗档案

| 项目 | 内容 |
|------|------|
| 出现房间 | F-09「雨衣下的祭坛」 |
| HP | 600 |
| 战斗时间预估 | 3-5 分钟（首通） |
| 阶段数 | 2 |
| 难度定位 | 入门级，玩家熟悉战斗节奏的考验 |
| 失败后 | 回到 F-13 残响碑 |

### 4.2 战场设计

- 圆形战斗区域，直径 12m
- 中央一根电线杆 + 摇晃的红灯笼（提供照明）
- 持续雨势，每 30 秒电闪一次（瞬间全屏白光）
- 可破坏掩体：4 个倒下的木箱（被攻击 5 次破坏）

### 4.3 阶段设计

#### 阶段 1（HP 100% → 50%）「找伞的人」

**节奏**：偏慢，给玩家学习时间。

**招式池**（随机选用）：
- ✦【折伞戳】：直线突刺，启动 0.7s，伤害 20，距离 4m
  - **破解**：侧滚或冲刺斩
- ✦【湿衣甩击】：360° 旋转挥击，启动 1.0s，伤害 25
  - **破解**：拉开距离
- ✦【伞砸地】：跳起后砸下，启动 1.2s，伤害 30，圆形 3m 震波
  - **破解**：跳跃前的影子提示，往外圈走
- ✦【喃语回响】：每 20 秒触发一次，全屏白光 0.5s（短暂遮蔽视野）
  - **解读**："记得带伞……"

#### 阶段过渡（HP 50% 触发）

- Boss 跪下，雨水从他身上倾泻
- 镜头拉近，他抬头：「车站三号口……别走错……」
- 战场中央出现一把**完整的红色小伞**——这是阶段 2 的核心机制

#### 阶段 2（HP 50% → 0%）「记起来的父亲」

**节奏**：中速，引入新机制。

**新机制：红伞之雨**
- 每 15 秒，整个战场会下"红色暴雨"持续 5 秒
- 红雨期间：玩家若**站在红伞下**则免疫所有攻击；若不在伞下，每秒受到 5 点伤害
- 但 Boss **会优先攻击伞**——伞被打掉就要等下次刷新

**新招式**：
- ✦【夺伞冲撞】：直线高速冲向红伞，伤害 30 并击碎红伞
  - **破解**：站在伞旁边引诱，他冲过来时侧闪
- ✦【伞骨投掷】：远程 3 连发飞射伞骨，每发 12 伤
  - **破解**：W 形走位
- ✦【失控旋转】（HP < 20% 触发）：原地疯狂旋转 5 秒
  - **破解**：拉远后蓄力斩 / 投掷物输出

### 4.4 击败演出

> 雨衣脱落，露出空空的内里。一把完整的红色小伞从空中飘落，缓缓落进他原来的位置。
> 雨停了。第一次有阳光透过云层。

**奖励**：
- 缠红丝技能
- 红茶蓝图（道具）
- 60 绯红丝缕
- 静默之心 ×1（满血上限永久 +25 残响值）

**招式池**：

  - **破解**：闪避或攻击拦截（字可被攻击破坏）
- ✦【纸页风暴】：召唤 6 张围绕场地飞行的纸页（持续 8 秒，碰到伤 5）
- ✦【墨点投掷】：往地面投掷墨水，制造小型墨水池

- 每 12 秒，Boss 会"提笔" → 屏幕上方出现 3 个红色文字（如「他」「她」「我」）

- 击破不同的字，Boss 下一段招式池不同（玩家可以选择"让 Boss 写什么"）

**新招式**：
- ✦【召唤白怨·翻页者 ×2】（中场抽援）
- ✦【挥笔重击】：启动 1s，挥下后留 3m 的红色墨痕（持续 5s 的伤害区）

#### 阶段 3（HP 33% → 0%）「写完了」

**新机制：终章**
- 整个战场被红墨水覆盖大半
- Boss 进入"狂书"状态——所有招式启动时间 -30%

**新招式**：
- ✦【END. 重锤】：屏幕显示巨大的"END."字样从天降，必须在 1.5s 内躲到字符外
  - **破解**：影渡冲出红色印章范围
- ✦【字海】（HP < 10% 触发）：召唤大量飞行字符填满屏幕，玩家必须找到唯一的安全角

### 7.3 设计哲学

> 这场 Boss 是给玩家的"成人礼"。Boss 4 使用阿茜的全部招式 —— 普攻三连、冲刺斩、缠红丝、影渡、甚至凝时斩。**玩家在战斗中学会的"读招"，将在这里被反过来考验**。

### 7.4 阶段设计

#### 阶段 1（HP 100% → 75%）「初心」

Boss 只用普攻 + 冲刺斩，不使用其他能力。
**招式池**：
- ✦【三连斩】：风/雪/月，与玩家招式一模一样
- ✦【冲刺斩】：横向突进，可接空中追击

**这一阶段是"对镜战"**——你用什么招，她也用什么招。

#### 阶段 2（HP 75% → 50%）「成长」

**新增**：缠红丝 + 影渡

**新招式**：
- ✦【红丝拽近】：远程钩爪 6m，命中将玩家拉至 Boss 身前
- ✦【影渡突击】：高速冲刺至玩家身前，附带斩击
- ✦【蓄力斩】：1.5s 蓄力 + 红色长光波（伤害 60）

#### 阶段 3（HP 50% → 25%）「绽放」

**新机制：红栀子绽放**
- 中央的红栀子完全盛开
- Boss 周身环绕红色花瓣，**任何玩家攻击会被反弹一次**（每 5s 重置）
- 玩家必须等花瓣消散的 1s 窗口期攻击

**新招式**：
- ✦【花瓣切割】：全场红色花瓣旋舞 3s（伤害 8/触碰）
- ✦【绯红霁】：高高跃起，落地时圆形震波（伤害 50）

#### 阶段 4（HP 25% → 0%）「凝时」

**Boss 解锁凝时斩。**

**新招式**：
- ✦【凝时斩 - 来袭】：当玩家攻击命中 Boss 时，Boss 25% 概率触发凝时斩反击
  - 玩家会**真正经历一次 Boss 视角的凝时斩** —— 全屏去色，慢动作看到 Boss 在跳跃，自己被斩 5-8 次
  - **唯一破解方式**：在 Boss 启动凝时斩瞬间（剑光闪起的 0.4s 内）按下空格 → 玩家弹反，**夺回主动权**
- ✦【最后一斩】（HP < 5% 触发）：Boss 冲向玩家做出致命一击，**唯一破解方式是弹反**
  - 弹反成功 → 进入击败演出
  - 弹反失败 → 玩家受重伤但不死，Boss 后退，再来一次

> 🎯 **关键设计**：玩家必须**在战斗中学会弹反**。每次 Boss 启动凝时斩，剑刃会闪起绯红光 0.4s —— 这就是窗口提示。第一次玩家可能会死好几次，但慢慢就会读招。

### 7.5 击败演出

> 详见第二卷 §5.4。
> 她跪下，剑插入地面。她抬头看向阿茜，露出第一次微笑。
> 红色融入阿茜身上。

**奖励**：
- 凝时斩（玩家终于学会）
- 静默之心 ×1（最后一颗，残响槽上限达到 200）
- 200 绯红丝缕

## 8. 隐藏 Boss：最初的咏（详细战斗设计）

### 8.1 战斗档案

| 项目 | 内容 |
|------|------|
| 出现房间 | E-04「最初的咏」 |
| HP | 1500 |
| 战斗时间预估 | 12-20 分钟 |
| 阶段数 | 3 |
| 难度定位 | 全游戏最难，**为二周目挑战玩家设计** |

### 8.2 战场设计

- 全黑空间，唯一光源是 Boss 手中的黑色灯笼
- 玩家与 Boss 之间有一条永远在静止流淌的白色河流
- **静音战**：BGM 完全消失，仅保留环境声 + 玩家自身脚步与剑响

### 8.3 设计哲学

> 这是关于"被遗忘的姓名"的战斗。她不是恶人，她是阿茜失去的一切。
> 所以她**模仿玩家的所有动作** —— 因为她原本就是阿茜的一部分。

### 8.4 阶段设计

#### 阶段 1（HP 100% → 66%）「无声」

**机制：动作模仿**
- Boss 会在玩家执行动作 0.5s 后做出**完全相同的动作**
- 玩家移动、攻击、闪避，她都会复刻
- **破解思路**：用"假动作"骗她（如假装攻击但及时取消，让她攻击空气）

**特殊规则：凝时斩反惩罚**
- 玩家每使用一次凝时斩，Boss HP 反而 +50（她变强）
- 强迫玩家用"普攻 / 道具 / 钩绳"等手段战斗

**招式池**（被动模仿玩家）：
- ✦ 镜像三连斩
- ✦ 镜像蓄力
- ✦ 镜像冲刺斩

#### 阶段 2（HP 66% → 33%）「呼唤」

**新机制：名字之河**
- 战场中央的白河开始浮现"被遗忘的字符"
- 玩家可以拾取字符（按 E）
- 拾取每个字符给阿茜回血 50，但 Boss 也会因此短暂变强
- 玩家可选择"读字符 vs 不读"

**新招式**：
- ✦【低语】：远程白色音波，伤害 25
- ✦【追忆斩】：Boss 突然进入"狂暴模式"，连续做 5 个高速动作，每动作伤害 20
- ✦【共鸣】：与玩家的招式同步释放，造成双倍伤害

#### 阶段 3（HP 33% → 0%）「记起来」

**新机制：名字归还**
- 玩家如果在阶段 2 拾取了 ≥5 个字符 → 触发"名字归还" → Boss 短暂呆滞 5s（巨大输出窗口）
- 玩家如果未拾取或拾取不足 → 直接进入硬战

**新招式**：
- ✦【最后的咏唱】：Boss 唱出阿茜的名字（无声字幕），全场震波伤害 50
- ✦【消散一击】（HP < 5% 触发）：Boss 自我消散，朝玩家扔出黑灯笼，必须用凝时斩弹反才能击败

**击败时**：
- BGM 突然回归（一段简单的钢琴旋律）
- 屏幕显示一行字幕：
  > 「她叫做——茜。」
  > （阿茜的真名）

### 8.5 奖励

- 「最初的红丝」（永久 +20% 攻击力，可装备到武器）
- 解锁结局 C
- 二周目专属皮肤：白色阿茜（服装与发色更接近残响白，仅左眼与剑柄红绳保留绯红）

## 1. 美术总览与设计语言

### 1.1 美术核心目标

> **让玩家在第一秒就明白：这是一个"褪色的世界"，而少女是仅存的颜色。**

### 1.2 美术三大支柱

**支柱一：清线动漫角色 × 2D 分层水彩场景**
- **角色层**：清晰线稿、细致发丝、瞳孔高光、偏写实动漫比例；角色必须能在透明背景上独立读取，便于直接作为游戏动作帧资产
- **场景层**：2D 手绘水彩 + 颗粒纹理，使用前景 / 中景 / 远景 / 氛围层的分层素材和轻微视差制造纵深
- **融合手法**：角色与场景之间不靠"硬描边"分离，而是通过**亮度对比 + 红色锚点**自然分层；所有图片生成都必须遵守同一套构图、色板与红色使用规则

**支柱二：水彩晕染 × 纸纹颗粒**
- 笔触基准：远景边缘湿润、低对比、轻微晕开；近景边缘干净、纹理清晰、能读出平台碰撞边界
- 关键技术：所有 2D 贴图叠加一层"水彩纸纹理 + 噪点 5-10%"，但角色动作帧的轮廓不能被颗粒破坏
- 直接生成要求：可用素材必须避免复杂光源、强投影、真实摄影皮肤、3D 塑料质感和不可拆分的背景角色混合

**支柱三：极简色彩主义**
- 全游戏色板严格控制：**95% 以上的画面是"褪色系"**（淡灰、浅青、米白、低饱和区域色），**唯一高饱和色只能是绯红 `#C8102E`**
- 区域辅助色只承担湿度、温度、材质和空间层次，不承担引导功能；引导、弱点、交互和记忆未褪色只能使用绯红
- 没有多余的颜色 = 没有多余的信息 = 玩家注意力始终被"红"引导

### 1.3 内部风格圣经

> 本项目的图片生成不再依赖外部作品名或作者名作为风格锚点。所有 AI 提示词必须从以下内部规则出发。

| 维度 | 生产规则 |
|------|---------|
| 角色线稿 | 清晰、干净、细线，五官克制，身体比例 1:6.5；动作帧必须有可读剪影 |
| 场景笔触 | 低饱和水彩、纸纹颗粒、远景湿边、近景干边，平台边缘必须清楚 |
| 镜头视角 | 纯 2D 横版侧视；角色动作用透明或纯色背景，场景用 16:9 分层横版构图 |
| 色彩控制 | 除 `#C8102E` 外不得出现高饱和强调色；红色面积在场景中 1-5%，角色中约 10% |
| 光影规则 | 柔和漫射光，避免霓虹、强彩色边光、写实体积光；红光只在残响、弱点和关键记忆中出现 |
| 资产可用性 | 每张图都要能归入资产清单：角色锚点、动作帧、Boss 帧、场景 key art、视差层、地形组件、UI/道具 |

### 1.4 视觉语言铁律

> 这些铁律不可妥协，是开发期所有美术资产审核的底线：

1. **画面中绝不能出现"红色之外"的高饱和色**（除非剧情演出特殊需要）
2. **每个房间至少 1 处红色锚点，但不超过 3 处**（多了会失去引导意义）
3. **角色阿茜的红色面积始终是画面中最显眼的**（即使在红色密集的房间也是）
4. **白怨敌人禁止使用任何红色**（除了"弱点"那一小撮）
5. **场景中的"褪色之壁"不是单纯的灰色，而是带极淡红色残影的灰色**——暗示这里曾有"红"

### 1.5 AI 直接生产原则

> 美术资源默认由图片生成承担第一版生产，而不是仅作为概念稿。人工修缮只用于清理边缘、拆分图层、统一尺寸与少量补帧。

**生产顺序**：
1. **角色锚点图**：先生成阿茜、咏、Boss 的正面 / 侧面 / 背面标准图，锁定服装、红色锚点、比例和剪影
2. **动作帧表**：按动作生成连续关键帧，要求透明背景、固定角色比例、红色拖尾单独可读
3. **地图组件库**：按六大区域生成 key art、房间原型、视差层、地形组件、交互件与红色锚点物
4. **批次验收**：每批按色彩、轮廓、比例、透明度、红色用途、可拆分性进行筛选

**资产清单字段**：
| 字段 | 用途 |
|------|------|
| `asset_id` | 唯一编号，例如 `AKN_ACT_attack_01` / `MAP_Forest_parallax_fg_01` |
| `类别` | 角色 / 动作 / Boss / 敌人 / 地图 / 组件 / UI |
| `用途` | 游戏内用途或文档用途 |
| `视角/比例` | 横版侧视、正面、背面、16:9 key art、1:1 icon 等 |
| `红色锚点` | 明确红色出现位置；没有红色时写"无" |
| `生成提示词` | 可直接用于 Codex 图片生成的中文结构化提示词 |
| `负面约束` | 禁止高饱和杂色、3D、摄影、文字、水印、错误红色等 |
| `验收标准` | 能否入库的判定标准 |
| `关联策划案章节` | 追溯到剧情、战斗或关卡设计来源 |

### 2.3 色板使用比例（每个画面构成的"配方"）

```
推荐比例（场景）：
  ├─ 区域中性色   60-70%   （主色板的灰白系）

推荐比例（角色）：
  ├─ 残响白       70%
  ├─ 墨灰 / 黑    20%
  └─ 茜色          10%   （丝带 / 瞳色 / 武器红绳）
```

> ⚠️ **铁律**：场景中的红超过 5% 时，需要美术总监审批。红是奢侈品。

## 3. 角色美术规格

### 3.1 阿茜（主角）

**基础信息**：
- 身高 158cm（与场景物件比例锚点）
- 头身比 1:6.5（偏写实动漫，不是 Q 版）
- 立绘分辨率：4K 原始稿，游戏内 1080p 缩放
- 帧动画：30fps（保留水彩"略有迟滞"的呼吸感）

**视觉细节**：
| 部位 | 规格 |
|------|------|
| 头发 | 银白短发（齐耳偏短），发丝逐根可见，左侧别绯红丝带（约 8cm 长，会随风飘） |
| 瞳色 | 右眼银灰（HEX `#A8A8B0`），左眼绯红（HEX `#C8102E`） |
| 服装 | 白色高领无袖短上衣 + 黑灰百褶短裙 + 白色及膝靴 + 红色细腰带 |
| 武器 | 单手细身长剑（剑长 70cm），剑柄缠红绳，剑刃微泛白光 |
| 表情 | 默认面无表情，仅在剧情高光点（Boss 4 击败 / H-08 / 终幕）有微妙变化 |

**动作设计哲学**：
- **战斗动作冷静干脆**——不要花里胡哨的甩剑回旋，每一招都"必要而精确"
- **走路有微妙拖步感**——表现"她也是残响"的伏笔
- **小动作要有**：拨头发、轻轻摸剑柄红绳、看到红色物件时停顿 0.3 秒——这些是塑造"反差感"的关键

**特殊状态视觉**：
| 状态 | 视觉变化 |
|------|---------|
| 残响态 | 全身渗出绯红光晕，残影拖尾 |
| 弹反凝时斩 | 左眼瞳孔放大，绯红色蔓延至半张脸 |
| 受伤 | 服装边缘微微"褪色"（向白色化）短暂 1 秒 |
| HP 低 | 剑柄红绳颜色变浅，呼吸有轻微抖动 |

### 3.2 渡舟人「咏」

**视觉要点**：
- 性别中性，看不清面容（始终被宽帽檐 + 围巾遮住）
- 身高比阿茜略高（约 168cm），略微驼背
- 全身灰白长袍，无任何红色（与"她是阿茜的遗忘"设定呼应）
- 手持一盏白色灯笼，灯火是**低饱和灰白冷光**，不能成为第二强调色
- 静态时几乎不动，仅头部偶尔转向阿茜

**关键演出**：
- 终幕咏消失时，灯笼留在原地，灯火渐渐转为白色
- 二周目时灯火变成**极淡绯红**（暗示她"知道阿茜回来过"，红色面积仍必须小于阿茜）

### 3.3 四位 Boss 美术规格

| Boss | 关键视觉 | 红色锚点 | 特殊动效 |
|------|---------|---------|---------|
| 雨衣的男人 | 褪色雨衣 + 折伞 + 看不见脸 | 阶段 2 出现的红色小伞 | 雨水从他身上"渗出而非滴落" |
| 未写完的人 | 上半身白衬衫 + 下半身飘散纸页 | 笔尖一滴永不落下的红墨 | 纸页有半透明感，能看见后面背景 |
| 灯塔守 | 厚重大衣 + 围巾遮面 + 红灯 | 手中红灯笼（全游戏最大的红） | 走动时灯光会在地面拖出长影 |
| 持剑的少女 | **与阿茜完全相同** | 与阿茜相同（左眼 + 红绳 + 腰带） | 战斗中红色逐渐"流向"阿茜 |

**击败演出共通规格**：
- 每个 Boss 击败时镜头拉近 2 秒慢动作
- 红色锚点物件**飘起 → 化作红光 → 融入阿茜**
- BGM 渐弱至完全静默 1 秒，然后切入区域终曲

### 3.4 普通敌人美术规格

**统一规格**：
- 全部白色基底（残响白 `#F5F1E8`）
- 边缘略带"褪色"晕染（向透明过渡）
- **唯一红色 = 弱点位置**，约占敌人面积 1-3%
- 动作偏迟缓，关节有"水彩晕染"的不稳定感
- 死亡时定格 0.3 秒后化作白色灰烬飘散

**8 种首发普通敌人 + 4 种备选普通敌人 + 4 种精英的视觉差异化**：
- 普通敌人体积统一 = 阿茜的 1.0-1.2 倍
- 精英体积 = 阿茜的 1.5-2.0 倍
- **精英有微弱红色光晕**（暗示"还没完全褪色"），普通敌人无光晕

## 4. 场景美术规格

### 4.1 场景制作流程

> 纯 2D 场景的制作流程建议（Godot 4.6）：
>
> 1. 关卡白盒（TileMapLayer + 简单碰撞，先验证跳跃距离）
> 2. 分层背景草图（远景 / 中景 / 前景各 1-2 层）
> 3. 绘制可复用 2D tile 与平台件（地面、墙、破壁、门、装饰）
> 4. 添加视差、雾、雨、纸灰等轻量粒子
> 5. 后处理：去色保红 Shader + 颗粒滤镜 + 水彩纸纹理叠加

### 4.2 场景共通规格

| 项目 | 规格 |
|------|------|
| 镜头视角 | 纯 2D 横版侧视 |
| 房间平均尺寸 | 约 3200 × 1800 px（Boss 房约 3840 × 2160 px）|
| Tile 基准 | 16 × 16 或 32 × 32 px 逻辑格，最终可用高清贴图覆盖 |
| 贴图分辨率 | 主体物 512-1024，背景层 1920 宽可循环，远景可降至 1024 |
| 灯光风格 | 以手绘明暗和 CanvasModulate 为主，避免大量实时点光源 |
| 后处理 | 去色保红 Shader / 颗粒 5% / 水彩纸纹理叠加 |
| 帧率目标 | 60fps（Mac M 系列芯片与中端 Windows 笔记本基准） |

### 4.3 各区域场景特色清单

#### 濡羽之森
- **必备元素**：黑色树干（要 5-7 种树形态）、湿润的地面反光、漂浮的水汽粒子、散落的红色伞骨道具
- **天气**：永夜雨，雨势从小到大变化，雷电仅在 Boss 战
- **音效锚点**：远处有水滴声 / 鸟鸣（频率：每 30-60 秒一次）

#### 燃尽书馆
- **必备元素**：成排被烧毁的书架、飘动的烧焦纸页、悬空的破碎玻璃天井
- **光影**：天井透下斜射光柱，光柱中可见纸灰浮动
- **特殊机制视觉**：飘动书页有半透明边缘，被攻击时会燃烧成红色灰烬

#### 沉钟海
- **必备元素**：搁浅的巨大钟楼（全场地标）、干涸的海床（裂纹纹理）、远处地平线的"船的剪影"
- **天气**：永远的暮色，地面有海星化石和锈链
- **远景**：全游戏最远的视野，能看到 200m 外的钟楼

#### 白栀庭
- **必备元素**：永不凋零的白色栀子（密集种植）、白色低墙、白色石板小径
- **光影**：全场最亮，白色叠白色制造"光晕过曝"效果
- **关键物件**：唯一一朵盛开的红栀子（Boss 房中央）

#### 中央枢纽
- **必备元素**：静止的忘川河、停泊的渡船（船身有红色纹路）、燃着红炭的铸炉、咏的小屋
- **特殊**：河面**不映出咏**——这是细心玩家可以发现的伏笔

#### 隐藏区域
- **必备元素**：全黑空间、白色河流（光源）、漂浮的白色文字粒子
- **特殊**：除黑灯笼外，画面中**没有任何光源**——是全游戏最压抑的视觉

## 5. UI 设计规格

### 5.1 UI 哲学：极简到近乎"没有"

> 玩家进入战斗后，**理想状态是 2 秒内忘记 UI 的存在**。

### 5.2 字体规格

| 用途 | 字体 | 字号 | 颜色 |
|------|------|------|------|
| 标题 / 章节标题 | 思源宋体 SemiBold（中）/ Marcellus（英）| 36-48pt | `#F5F1E8` |
| 正文 / 对白 | 思源宋体 Regular（中）/ EB Garamond（英）| 18-22pt | `#F5F1E8` |
| 阿茜内心独白 | 思源宋体 Italic 倾斜 | 20pt | `#E8B4B8`（樱褪色） |
| Boss 喃语 | 手写感字体（如 Klee One）| 24pt | `#FAFAF7` 半透明 80% |
| 数值 / 数字 | DIN Condensed | 14-16pt | `#F5F1E8` |
| 残响碎片文字 | 手写感字体 | 18pt | `#C8102E` |

> ⚠️ **铁律**：游戏内任何文字**不出现"系统感的等宽字体"**（如 Consolas）——会破坏水彩氛围。

### 5.3 HUD 元素规格

```
左上角 · HP 条
  ├─ 形态：白色细线（粗 2px），长度 200px
  ├─ 受损时：受损部分变绯红 + 短暂震动 0.2s
  ├─ 满血时：完全消失（看不见）
  └─ HP < 30% 时：边缘渗出红色雾化边

左上角下方 · 残响槽
  ├─ 形态：圆环（直径 60px，线宽 3px）
  ├─ 蓄槽时：圆环按比例填充绯红色
  ├─ 满槽时：圆环缓慢顺时针旋转 + 微脉冲
  └─ 触发残响态时：圆环爆开成粒子 + 0.5s 闪光

右上角 · 道具栏（4 槽）
  ├─ 形态：4 个小方框（24×24px），白色细线边框
  ├─ 槽内显示道具图标（手绘风）
  ├─ 数量在右下角小字 DIN Condensed 12pt
  └─ 使用中道具：图标短暂放大 1.2 倍 + 红色描边

中下方 · 互动提示
  ├─ 形态：仅在 E 互动可用时浮现"轻轻按 E"白色文字
  ├─ 出现 0.3s 渐入，消失 0.5s 渐出
  └─ 永不停留超过 5 秒
```

### 5.4 菜单与地图

**主菜单**：
- 背景：水彩晕染的茜色（左下到右上渐变）
- 选项纯文字（中央对齐），鼠标悬停时**仅文字颜色变红**，无背景框
- 选项："开始 / 继续 / 设置 / 退出"——绝不超过 5 项

**地图界面**：
- 黑色底（`#0F0F12`）+ 白色细线绘制房间
- 已探索房间：白色实线
- 未探索房间：白色虚线（仅显示与已探索房间相邻部分）
- 残响碑：白色小圆点，可传送的会有微脉冲
- 当前位置：唯一一个红色小点
- **不显示残响碎片位置**——保持探索神秘感

**道具栏（TAB 键打开）**：
- 半透明黑色蒙层（不暂停游戏，营造紧迫感）
- 4 个装备槽 + 12 个可选道具列表
- 切换时有"水彩晕染"过渡动画（0.3 秒）

### 5.5 UI 动效规格

| 动效 | 时长 | 缓动 |
|------|------|------|
| 文字浮现 | 0.3-0.5s | ease-out |
| 文字消失 | 0.5-0.8s | ease-in |
| HUD 元素脉冲 | 1.5s 周期 | sine 平滑 |
| 菜单切换 | 0.3s | ease-in-out |
| 地图打开 | 0.4s | ease-out + 颗粒过渡 |

### 5.6 字幕规格（关键设计）

阿茜内心独白与 Boss 喃语的展示方式 **不同**：

**阿茜内心独白**：
- 屏幕底部居中，小字（20pt），樱褪色
- 一字一字"渗墨"出现（每字 0.05s）
- 整段读完后停留 2 秒消失
- **没有"对话框边框"**，直接漂浮在画面中

**Boss 喃语**：
- 屏幕中央偏上，更小的字（18pt），半透明白色
- 字符位置略微抖动（不稳定感）
- 战斗中持续浮现，营造"她还没死透"的怨念感

## 7. 音乐总览与定位

### 7.1 音乐核心目标

> **沉默与音乐同样重要。**这不是"BGM 不停的游戏"，而是"音乐恰到好处出现"的游戏。

### 7.2 音乐风格定位

**主风格**：极简后摇 / 钢琴 + 弦乐 + 微量电子氛围

**参考作品**：
- 《GRIS》— Berlinist（**最贴近本作的参考**）
- 《Journey》— Austin Wintory
- 久石让的内省作品（钢琴 + 室内乐）
- Joep Beving 的极简钢琴

**绝不使用的风格**：
- 高强度电子 / 鼓点驱动的 EDM
- 管弦乐宏大叙事（如《魔兽世界》主题）
- J-RPG 式的"战斗 BGM 循环"

### 7.3 音乐三层结构

```
【环境层】 ─ 几乎听不见，仅创造氛围（风声 / 钢琴单音 / 弦乐持续音）
【主旋律层】 ─ 关键时刻浮现，承载情感（钢琴 / 单簧管 / 大提琴独奏）
【冲击层】 ─ Boss 战与情感高峰使用（弦乐齐奏 / 强力打击 / 极少使用）
```

> 80% 的游戏时间只有【环境层】，20% 才有主旋律加入，**冲击层全游戏不超过 6 次**。

### 7.4 沉默时刻清单（不放音乐的关键时刻）

| 时刻 | 处理 |
|------|------|
| H-06 时间静止之屋 | 仅环境音（粉笔灰落地 / 远处钟摆声）|
| H-08 学生的房间真相揭示 | 完全静音，仅心跳声 |
| 隐藏 Boss「最初的咏」全战 | BGM 完全消失，仅自身脚步与剑响 |
| 三种结局开场前 5 秒 | 静默 |
| 玩家死亡后回到残响碑 | 静默 3 秒，再缓慢淡入区域 BGM |

### 8.2 关键音乐时刻设计

#### 主题曲《茜》的旋律动机
- 一段简单的 6 音钢琴动机（如 La-Mi-Re-Do-Mi-La），全游戏多个曲目变奏使用
- 玩家在不同情境下听到这个动机，会逐渐建立**"这是阿茜本人的音乐"**的印象
- 终幕时主题曲完整版响起 → 情感峰值

#### Boss 4《送我》的设计要点
- **Boss 4 的前 3 阶段是激烈战斗音乐《对镜》**
- **阶段 4 切入凝时斩时，音乐骤然变成《送我》钢琴独奏**
- 这种"激烈 → 简化"的反差，会让玩家在最难的战斗中突然感到悲伤
- 这是游戏最重要的音乐设计点，**不可妥协**

#### 结局 C《老师，您回来了》的设计要点
- 仅在结局 C 出现
- 包含一段**少年的哼唱**（无歌词，纯哼）—— 这是全游戏唯一的"人声"
- 哼唱的旋律就是主题曲《茜》的动机
- 暗示：少年一直在哼着记忆中老师哼过的曲子

### 8.3 动态音乐系统

> 这是为提升沉浸感的关键技术：

**层级触发**：
- 房间无敌人 = 仅【环境层】
- 房间有敌人但未战斗 = 【环境层】+ 微弱【主旋律层】渐入
- 战斗中 = 完整【主旋律层】
- Boss 阶段切换 = 切到【冲击层】30 秒后回落

**淡入淡出规则**：
- 进入新区域：旧 BGM 1.5 秒淡出 → 静默 1 秒 → 新 BGM 2 秒淡入
- Boss 战开始：现有 BGM 0.5 秒急停 → Boss BGM 立即开始
- Boss 击败：BGM 渐慢 → 最后一音延长 → 静默 1 秒 → 区域 BGM 回归

## 9. 音效设计

### 9.1 音效设计哲学

> **少而准。**每个音效都该让玩家"听懂含义"，不是单纯的"装饰声"。

### 9.2 战斗音效清单

| 类别 | 音效 | 设计要点 |
|------|------|---------|
| 普攻命中 | 风（高频） / 雪（中频） / 月（低频）| 三连斩三段音色不同，建立节奏感 |
| 蓄力斩 | 蓄力期低频嗡鸣 + 释放高频"裂"声 | 蓄力期间持续音提示玩家进度 |
| 冲刺斩 | 短促"嗖"+ 命中"砰" | 与影渡的音效不同，避免混淆 |
| 影渡 | 仅"嗖"，无命中音 | 与冲刺斩对比，让玩家区分 |
| 弹反成功 | 金属高频"叮" | 全游戏最清脆的声音，是奖赏感来源 |
| 凝时斩开始 | 全场音效抽离 + 钟声"咚" | 暗示"时间停了" |
| 凝时斩中每次斩击 | 低频"咔" | 时停期间唯一的声音 |
| 凝时斩结束 | 全场音效骤回 + 低频"咚" | 与开始呼应 |
| 受击 | 阿茜的"啊"（极小声）+ 闷响 | 阿茜的受击音是全游戏唯一的"她的声音" |
| 死亡 | 长音渐弱 + 心跳停 | 强烈的"消散"感 |

| 区域 | 关键环境音 |
|------|-----------|
| 濡羽之森 | 雨声（持续）/ 滴水（30s 一次）/ 远处鸦鸣（60s 一次）/ 雷声（仅 Boss 战）|
| 燃尽书馆 | 纸页翻动（持续低音）/ 木材吱呀（偶尔）/ 火苗噼啪（焚烧之厅）|
| 沉钟海 | 海风（持续）/ 远处钟声（60s 一次）/ 锈链摩擦（钟摆区）|
| 白栀庭 | **几乎完全静默**（仅风声极弱）/ 远处教学铃（H-06 触发） |
| 中央枢纽 | 河水流动（持续）/ 木船吱呀（偶尔）/ 铸炉炭火（铸炉区） |

|------|------|

## 10. 配音指引

### 10.1 配音原则

> **配音 ≠ 念稿。**本作配音以"气息感"为核心，文字精简但表演要重。

### 10.2 阿茜的内心独白（核心配音工作）

**配音定位**：
- 首发可完全不配音；若预算允许，优先只录中文阿茜独白与 Boss 关键喃语
- 英文版首发建议仅文字本地化，不强制英文配音
- 中文优先选择**清冷少女声 + 偏中性**的女声优
- 参考声线：花泽香菜的低音区 / 户松遥的冷静线 / 早见沙织的内省感
- **绝不使用过度可爱 / 萝莉化的声线**——会破坏反差感

**演绎要点**：
- 阿茜的独白都是"自言自语"，不是"对玩家说"——配音时假设没人在听
- 抽象幽默的句子要"平淡读"而非"搞笑读"——反差感来自语调的克制
- 关键句子（如 H-08"我想起来了。全部。"）需要**长时间停顿**，气息感胜过台词

**示例处理**：

> 「这棵树长得像我室友。我没有室友。」
>
> ❌ 错误读法：先卡通可爱地说"长得像我室友"，再吐槽"我没有室友"（笑点感）
>
> ✅ 正确读法：两句都用同一种平淡语气，仿佛在陈述事实（反差感）

> 「我以为我会哭。但其实没什么好哭的。」
>
> ❌ 错误读法：第一句颤抖 / 哽咽，第二句释然
>
> ✅ 正确读法：两句都是平静的，仿佛她在告诉自己一件已经接受的事

### 10.3 Boss 喃语配音

**Boss 配音特殊要求**：
- 所有 Boss 喃语**带回响 + 加水下感处理**（reverb 30% + 低频补偿）
- 4 个 Boss 的声线必须明显区分

| Boss | 声线参考 |
|------|---------|
| 雨衣的男人 | 中年男声，疲惫，略带方言（如带北方腔的中年大叔） |
| 未写完的人 | 青年男声，急促，神经质（像熬夜赶稿的作家） |
| 灯塔守 | 老年女声，沙哑温柔，不悲伤（接受了等待） |
| 持剑的少女 | 与阿茜**完全相同的声优**，但更冷、更年轻 |

### 10.4 隐藏 Boss "最初的咏"

- 配音方式：**仅在击败时说一句话**——阿茜的真名"茜"
- 处理：低声、仿佛叹息、与阿茜本人声音有"重叠感"（暗示她是阿茜的一部分）
- **音效后期处理**：加 0.5 秒前置静音 + 0.3 秒后置回响

### 10.5 配音工作量预算

| 角色 | 台词条数 | 总录制时长 |
|------|---------|----------|
| 阿茜（中文，可选） | 约 32 处独白 | 约 5 分钟成片 |
| 4 位 Boss（可选） | 各 4 句喃语 + 1 段击败前独白 | 约 3 分钟成片 |
| 隐藏 Boss | 1 句台词 | 约 5 秒 |
| **总计** | — | **约 8 分钟成片** |

> 折合录音棚工作时间：中文约 1 天（含 NG 与重录）。小预算版本建议全部改为文字 + 呼吸 / 叹息音效。

### 10.6 配音外包建议

**优先方案**：
- 中文：阿茜建议起用**新人声优 + 资深导演**组合（避免观众"听见熟人"出戏）
- 英文：首发只做字幕；销量验证后再补录

**备选方案**（小预算）：
- 全游戏改为**仅文字、零配音**（参考《空洞骑士》《Inside》）
- 此方案下，咏的"无声"和阿茜的"无声"形成统一美学，并不会损害体验
- **省下的预算建议投入 BGM 制作**

# 第六卷：AI 实现路线与验收标准

> 📌 本卷是给 AI 编程执行用的落地说明。它不增加新玩法，只规定实现顺序、房间规范与验收标准，确保这个游戏能被一步步做出来。

---

## 1. 可玩性优先级

所有实现任务按以下顺序排优先级：

| 优先级 | 内容 | 验收问题 |
|--------|------|----------|
| P0 | 移动、跳跃、攻击、受击、死亡、存档 | 不看美术，角色是否已经好控制？ |
| P0 | 残响槽、残响态、破壁 | 空格键三用是否真的产生取舍？ |
| P1 | 影渡、缠红丝、静音步、凝时斩 | 每个能力是否同时服务战斗和探索？ |
| P1 | 房间切换、地图记录、残响碎片 | 玩家是否知道自己去过哪里、还缺什么？ |
| P2 | Boss、精英敌人、结局 | 是否有完整通关压力与情感回报？ |
| P3 | 最终美术、配音、新游戏 + | 是否增强体验，而不是阻塞完成？ |

> 规则：P0 不好玩时，不做 P2；移动手感不成立时，不扩关卡。

## 2. 系统实现顺序

### 2.1 M1：手感原型

最小可玩范围：
- 1 个测试房间
- 阿茜占位图
- 1 个徘徊者敌人
- 1 面褪色之壁
- 1 个残响碑

必须完成：
- 左右移动、可变高度跳跃、土狼时间 0.1 秒、跳跃缓冲 0.1 秒
- 普攻三连、蓄力斩、冲刺斩
- HP、受击无敌、死亡回存档点
- 攻击命中增加残响，残响满后可进入残响态
- 残响 ≥50% 时可破壁

通过标准：
- 连续玩 10 分钟，移动和攻击没有明显卡手
- 玩家能自然理解"攒槽打怪"和"花槽破墙"的取舍

### 2.2 M2：濡羽之森垂直切片

必须完成：
- 忘川渡口 5 房间
- 濡羽之森 9 房间
- 影渡、缠红丝
- 2 种普通敌人 + Boss 1
- 9 个残响碎片

通过标准：
- 从新游戏开始到击败 Boss 1，全程无需开发者指路
- 首次游玩 20-30 分钟内完成
- 至少有 1 次玩家主动回头拿之前够不到的碎片

### 2.3 M3：全主线白盒

必须完成：
- 四大区域全部白盒连通
- 四个主线 Boss 均可击败
- 48 个残响碎片可收集
- 三结局逻辑可触发

通过标准：
- 从开局到结局 A/B/C 均可完整跑通
- 没有永久卡死、不可回收碎片、无法返回枢纽的问题

## 3. 房间制作规范

每个房间必须包含：
- 1 个入口和 1 个出口，Boss / 叙事房可例外
- 至少 1 个视觉锚点；普通房间不超过 2 个红色锚点
- 1 个明确目的：教学、战斗、解谜、叙事、回溯奖励五选一
- 清晰的碰撞边界，禁止依赖复杂斜坡和任意曲面
- 房间名、区域 id、门 id、存档点 id、碎片 id 都写入可读配置

房间禁止事项：
- 禁止同一房间同时引入两个新机制
- 禁止需要像素级跳跃
- 禁止无法预判的一击必死
- 禁止只有装饰、没有玩法或叙事作用的长走廊

## 4. 验收标准

### 4.1 好玩标准

| 项目 | 通过线 |
|------|--------|
| 移动 | 玩家能在 1 分钟内熟悉跳跃、落地、转身 |
| 战斗 | 普通敌人 3-6 秒解决，精英 30-60 秒解决 |
| Boss | Boss 1 首通 0-3 死，Boss 4 首通 5-12 死 |
| 探索 | 每 3-5 个房间至少看到一个"现在拿不到，以后回来"的目标 |
| 残响 | 玩家每 5 分钟至少做一次"用来打还是用来开路"的选择 |
| 叙事 | 每个区域通关后，玩家能说出这个区域的情绪主题 |

### 4.2 可实现标准

| 项目 | 通过线 |
|------|--------|
| 代码 | 单个核心脚本尽量不超过 300 行；超过则拆为组件 |
| 关卡 | 单个房间白盒制作不超过 1 天，美术替换不超过 0.5 天 |
| 敌人 | 普通敌人共享 EnemyBase，只覆写移动和攻击 |
| Boss | 每个 Boss 先做 2 个核心招式，通过后再加第 3 个 |
| 美术 | 所有最终美术缺失时，占位图也能完整通关 |
| 发布 | macOS 与 Windows 导出均能从主菜单进入并完成结局 A |

### 4.3 砍内容规则

当进度失控时，按以下顺序砍：
1. 新游戏 +
2. 隐藏 Boss「最初的咏」
3. 精英敌人数量，从 4 只砍到 2 只
4. 残响碎片，从 48 片砍到 36 片，并同步结局阈值
5. 配音全部取消，仅保留文字与音效

绝不能砍：
- 红色唯一性
- 残响槽的战斗 / 探索取舍
- 阿茜也是残响的真相
- Boss 4 学会凝时斩的成人礼
