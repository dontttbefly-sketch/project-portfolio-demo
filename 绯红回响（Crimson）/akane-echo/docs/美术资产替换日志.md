## 批次 2：second_pass_complete

**目标**：把批次 1 的通用图继续拆成可运行的专属资产，覆盖当前项目里的阿茜动作、普通敌人、精英敌人、Boss 与六区地图 key art。

**生成方式**：`tools/generate_complete_assets.py`

**资产清单**：`res://assets/generated/second_pass_manifest.json`

**替换范围**：

| 类别 | 新资产 | 使用位置 |
|------|--------|----------|
| 阿茜动作 | `characters/akane_actions/akane_*_second_pass.png` | `scripts/v2/Player2D.gd`、兼容旧 `Player.gd` / `Player.tscn` |
| 普通敌人 | `enemies/enemy_*_second_pass.png` | 当前所有普通敌人场景逐一替换 |
| 精英敌人 | `enemies/elite_*_second_pass.png` | 4 个精英敌人场景逐一替换 |
| Boss | `bosses/boss_*_second_pass.png` | 6 个 Boss 场景逐一替换 |
| 六区地图 | `environment/maps/map_*_key_second_pass.png` | `M1Prototype` 背景与后续区域接入常量 |

**运行时接入**：
- `Player2D` 已按当前状态切换动作图：待机、行走、跳起、下落、落地、三段斩、蓄力、冲刺斩、影渡、缠红丝、残响态、受击、死亡。
- `V2Assets` 保留兼容常量，同时新增动作、敌人、精英、Boss、六区地图的专属常量。
- `M1Prototype` 的背景从标题氛围图切换为忘川渡口 key art。

**验收重点**：
- 场景与脚本中不再引用 `enemy_white_first_pass`、`enemy_elite_first_pass`、`boss_faded_umbrella_first_pass` 或旧阿茜 `assets/characters/*.png`。
- 普通敌人与咏不使用大面积红色，只保留敌人弱点、交互件与阿茜核心红色锚点。
- 所有二批透明角色 / 敌人 / Boss 图保持透明背景，地图 key art 使用不透明横版背景。

**本批验收记录**：
- Godot headless import 已完成，47 张二批 PNG 均生成 `.import`。
- `res://` 引用完整性检查：`missing_refs = 0`。
- 透明角色 / 敌人 / Boss 四角检查：`transparent_corner_fail = 0`。
- 色彩检查：`non_crimson_high_saturation = 0`。
- 旧通用美术引用检查：场景与脚本中无 `enemy_white_first_pass`、`enemy_elite_first_pass`、`boss_faded_umbrella_first_pass`、旧阿茜 `assets/characters/*.png`。
