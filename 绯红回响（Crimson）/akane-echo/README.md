# 茜响 / Akane's Echo

> 一位以「绯红」为唯一颜色的少女，在褪色的「残响之境」中以剑斩开记忆。
> 纯 2D 横版类银河城 · macOS 首发 + Windows · 主线 90–120 分钟 / 全收集 2.5–3 小时

## 如何打开项目

1. 启动 Godot 4.6
2. 在项目管理器点击 **Import**（导入）
3. 选择本目录下的 `project.godot`，点击 **Import & Edit**
4. 第一次打开 Godot 会扫描资源（约 10–30 秒）
5. 进入编辑器后，左下角文件浏览器中双击 `scenes/main/MainMenu.tscn` 打开主菜单
6. 按 **F5** 运行游戏（首次会让你设置主场景，选 MainMenu.tscn 即可，不过我已经在 project.godot 里配好了）

## 开发约定

设计三原则（来自策划案 §1.3，所有决策的判断依据）：

1. **「红是唯一的颜色」** — 美术 / UI / 关卡引导都围绕此核心，绯红 `#C8102E` 是世界中唯一的高饱和色。
2. **「每个键都做两件事」** — 攻击 → 蓄槽，资源 → 战斗 + 探索两用，能力 → 战斗 + 解谜两用。
3. **「沉默胜过言语」** — 文字精简至骨，靠场景与残响碎片说话。
